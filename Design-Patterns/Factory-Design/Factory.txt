comes under creational design pattern

Application:
widely used in JDK(valueOf() method  in wrapper class like Boolean, Integer etc, and java.util.Calendar, Resource Bundle and Number Format getInstance() method uses Factory design) as well as frameworks like Spring and Struts

Principle:
used when we have as superclass with multiple sub classes and based on input, we need to return one of the sub class

Important points: 
We can keep the factory class Singleton or we can keep the method that returns the subclass as static 

Advantages: 
makes code less coupled hence easy to extend
provides abstraction between implementation and client classes through interface
